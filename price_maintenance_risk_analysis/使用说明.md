# 定增风险分析报告生成器 - 使用说明

## 🚀 快速开始

### 一键生成报告

```bash
# 生成华勤技术的报告
./gen_report.sh 603296.SH

# 生成光弘科技的报告
./gen_report.sh 300735.SZ

# 指定输出文件名
./gen_report.sh 603296.SH 华勤技术分析报告.docx
```

## 📝 首次使用配置

### 第一步：运行脚本
```bash
./gen_report.sh 603296.SH
```

### 第二步：脚本会自动创建配置文件模板
配置文件位置：`data/603296_SH_placement_params.json`

### 第三步：编辑配置文件
```bash
# 使用VS Code（推荐）
code data/603296_SH_placement_params.json

# 或使用nano
nano data/603296_SH_placement_params.json

# 或使用vim
vim data/603296_SH_placement_params.json
```

### 第四步：填写必填参数
打开配置文件后，填写以下必填参数：

```json
{
  "issue_price": 25.0,           // 发行价（元/股）- 必填
  "financing_amount": 1000000000, // 融资金额（元）- 必填
  "lockup_period": 6,             // 锁定期（月）- 必填
  "current_price": 28.5,          // 当前股价（元/股）- 必填
  "price_source": "定增预案",      // 定价依据说明

  // 以下为可选参数，建议填写以获得更准确的分析
  "net_assets": 15000000000,      // 净资产（元）
  "total_debt": 8000000000,        // 总债务（元）
  "net_income": 1200000000,        // 净利润（元）
  "revenue_growth": 0.15,          // 营收增长率（15%）
  "operating_margin": 0.12,        // 营业利润率（12%）
  "volatility": 0.35,              // 波动率（35%）
  "beta": 1.0                      // 贝塔系数
}
```

### 第五步：重新运行脚本
```bash
./gen_report.sh 603296.SH
```

## 📋 参数说明

### 必填参数

| 参数 | 说明 | 示例 | 数据来源 |
|------|------|------|----------|
| **issue_price** | 定增发行价格 | 25.0 | 定增公告/预案 |
| **financing_amount** | 融资金额 | 1000000000 | 定增公告/预案 |
| **lockup_period** | 锁定期（月） | 6 | 定增公告/预案 |
| **current_price** | 当前股价 | 28.5 | 实时行情（会自动更新） |

### 可选参数（建议填写）

| 参数 | 说明 | 示例 | 数据来源 |
|------|------|------|----------|
| **net_assets** | 净资产 | 15000000000 | 最新财报 |
| **total_debt** | 总债务 | 8000000000 | 最新财报 |
| **net_income** | 净利润 | 1200000000 | 最新财报 |
| **revenue_growth** | 营收增长率 | 0.15（15%） | 历史数据 |
| **operating_margin** | 营业利润率 | 0.12（12%） | 历史数据 |
| **volatility** | 波动率 | 0.35（35%） | 可自动计算 |
| **beta** | 贝塔系数 | 1.0 | 可自动计算 |

## 📊 报告内容

生成的报告包含以下章节：

1. **项目概况** - 基本信息、发行参数、投资亮点
2. **相对估值分析** - PE、PB相对行业分析
3. **DCF估值分析** - 现金流折现估值
4. **敏感性分析** - 参数敏感性测试
5. **蒙特卡洛模拟** - 10,000次模拟风险分析
6. **情景分析** - 多窗口期分析
7. **压力测试** - 极端情景测试
8. **VaR风险度量** - 风险价值分析
9. **综合评估** - 风险评分和投资建议
10. **风控建议** - 盈亏平衡分析和风控策略

1. 访问：https://tushare.pro/register
2. 注册账号
3. 在用户中心获取Token

## ❓ 常见问题

### Q1: 提示"配置文件不存在"？
**A:** 首次运行会自动创建模板，请编辑后重新运行

### Q2: 提示"必填参数未填写"？
**A:** 请检查配置文件中的以下参数：
- issue_price（发行价）
- current_price（当前股价）
- financing_amount（融资金额）
- lockup_period（锁定期）

### Q3: 报告生成失败，提示数据错误？
**A:**
1. 检查是否设置了TUSHARE_TOKEN
2. 运行时加上 `--force-update` 强制更新数据
3. 检查股票代码格式是否正确

### Q4: 如何获取上市公司的财务数据？
**A:**
- 巨潮资讯网：http://www.cninfo.com.cn
- 东方财富网：https://www.eastmoney.com
- 同花顺：https://www.10jqka.com.cn
- 上市公司官网投资者关系栏目

### Q5: 定增参数在哪里找？
**A:**
- 巨潮资讯网 → 公告 → 定增预案/公告
- 东方财富网 → 数据 → 定增中心
- 上市公司公告
